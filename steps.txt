To run the app:

Clone the Repo:
    git clone https://github.com/lokesha01hp/AI-Based-Virtual-Interview-Preparation-Assistant

Firstly,
    Install Python 3.10.11 (for the resume app), and Python 3.12.10 (for the remaining apps)

    Install the requirements
        pip install -r requirements.txt
    
    Create a virtual env inside the resume folder as:
        cd resume
        python3.8 -m venv env
        env/Scripts/activate
        pip install -r requirements.txt
    

Then,
    Open 5 Seperate Terminals to run 5 seperate app.py.
        Terminal 1:
            cd Aptitude_Generator
            python app.py
        
        Terminal 2:
            cd Interview_Assistant
            python app.py
        
        Terminal 3:
            cd Interview_Assistant
            python evaluator.py

        Terminal 4:
            cd resume
            env/Scripts/activate
            python app.py
        
        Terminal 5:
            cd main-app
            python app.py

After all the apps start
visit http://127.0.0.1:5000/
